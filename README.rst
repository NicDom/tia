Installation
------------

You can install *TIA* via pip_ from PyPI_:

.. code:: console

   $ pip install tia

or install with ``Poetry``

.. code:: console

   $ poetry add tia

Then you can run

.. code:: console

   $ tia --help

or with ``Poetry``:

.. code:: console

   $ poetry run tia --help

Uninstall:
~~~~~~~~~~

If you wan to uninstall the package, simply run

.. code:: console

   pip uninstall tia

or with ``Poetry``:

.. code:: console

   poetry remove tia
